## ğŸ›  Setup

1. Clone the repo:

```bash
git clone https://github.com/selenozcan/mixgenie.git
cd mixgenie
```

2) Install dependencies:

```bash
npm install
```

3) Set up your `.env` (for Groq key):

```bash

```

4) Run the app locally

```bash
npx expo start
```

You can preview with QR code via Expo Go.
